# Social Shorts Generator

Backend per Agente AI Social - genera shorts verticali da video YouTube.

## Features
- Download video YouTube (yt-dlp)
- Vertical clips 9:16 (ffmpeg)
- Upload Cloudflare R2

## Endpoint
- GET /health
- POST /process-social-video

## Deploy Railway
1. Connect GitHub repo
2. Add environment variables
3. Deploy!
